# homebrew-tap

A [Homebrew](https://brew.sh) tap for my macOS apps and tools.

## Install

```sh
brew install --cask anvaratayev/tap/rdio
```

That command adds the tap automatically. Alternatively, tap first and then
install by short name:

```sh
brew tap anvaratayev/tap
brew install --cask rdio
```

## Casks

| Cask | Description |
| --- | --- |
| [`rdio`](Casks/rdio.rb) | Tiny menu bar radio player — [source](https://github.com/AnvarAtayev/rdio) |
